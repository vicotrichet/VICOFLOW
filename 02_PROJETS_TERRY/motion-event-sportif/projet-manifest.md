# Manifest Projet : Motion Event Sportif

## Identité projet
- **Nom interne** : motion-event-sportif
- **Client final** : SwissRun Festival
- **Type mission** : Motion design pour écrans LED événementiels

## Contexte business
Course urbaine 10km à Lausanne (5000 participants attendus). Client veut habillage visuel ligne départ/arrivée + boucles vidéo écrans LED village partenaires.

## Livrables attendus
1. **Boucle vidéo ligne départ** (60 sec, 4K, 16:9)
   - Countdown dernières 10 secondes
   - Logo sponsors intégrés
2. **Boucle vidéo ligne arrivée** (90 sec, 4K, 16:9)
   - Classement live (API fournie par client)
   - Replay passages marquants
3. **Boucles village partenaires** (30 sec chacune, Full HD, 9:16 vertical)
   - 5 sponsors principaux
   - Format écrans LED verticaux

## Contraintes techniques
- **Formats** : MP4 H.264, 60fps minimum
- **Logiciels imposés** : After Effects + Premiere Pro
- **Délai rendu** : 48h avant événement (validation client)
- **Références visuelles** : 
  - Nike Run Club aesthetics
  - Red Bull motion graphics events

## Budget & deadline
- **Budget total projet** : 18'000 CHF
- **Budget freelance motion** : 8'500 CHF (TJM 850 CHF x 10 jours)
- **Deadline finale** : 15 mai 2026
- **Jalons** :
  - J+3 : Moodboard + directions créatives
  - J+7 : Animatiques validées
  - J+12 : Version finale livrée
